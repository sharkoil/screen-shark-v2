Common Issues:

  • Extension not detected: Make sure Screen Shark is installed and enabled
  • Content script not loaded: Reload the page and try again
  • Permission denied: Check extension permissions in chrome://extensions
  • Tests failing: Open browser console (F12) for detailed error messages

If extension is not working:

  1. Go to chrome://extensions
  2. Find Screen Shark extension
  3. Click "Reload" button
  4. Reload this page
  5. Try running tests again
